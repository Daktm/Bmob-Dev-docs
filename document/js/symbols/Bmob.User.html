Class Bmob.User

A Bmob.User object is a local representation of a user persisted to the Bmob cloud. This class is a subclass of a Bmob.Object, and retains the same functionality of a Bmob.Object, but also extends it with various user specific methods, like authentication, signing up, and validation of uniqueness.


Defined in: bmob.js.

{Bmob.Promise} signUp(attrs, options)
Signs up a new user. You should call this instead of save for new Bmob.Users. This will create a new Bmob.User on the server, and also persist the session on disk so that you can access the user using current.

A username and password must be set before calling signUp.

Calls options.success or options.error on completion.

Parameters:
{Object} attrs
Extra fields to set on the new user, or null.

A Backbone-style options object.
