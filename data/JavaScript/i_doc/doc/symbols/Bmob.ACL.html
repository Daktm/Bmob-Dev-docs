Class Bmob.ACL

权限控制可以被添加到任何 Bmob.Object，用来控制用户的访问权限


Defined in: bmob.js.

Class Summary
Constructor Attributes Constructor Name and Description
 
Bmob.ACL(arg1)
创建ACL 如果传任何参数，则任何人都没有权限 如果传入的参数是Bmob.User，那个usr会有读写权限。 如果传入的参数是json对象，则会有相应的acl权限。
